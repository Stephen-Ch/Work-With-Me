# Contributing to Work With Me

This project is closed to new feature work — see the [Status](README.md#status) section in the README for why.

## Bug reports

If you spot a real bug, an issue is welcome. Please include steps to reproduce.

## Forking

The idea and code are MIT-licensed. If you want to take the concept further — expand the questions, target a different audience, build in a different stack — feel free to fork it and run with it.

## Local development

- Install dependencies: `npm install`
- Start local app: `npm start`
- Run tests: `npm test`
- Build for production: `npm run build`
