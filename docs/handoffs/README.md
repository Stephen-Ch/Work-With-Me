## Handoff Guidelines

When creating new handoffs:
1. Date the filename (handoff-YYYY-MM-DD-topic.md)
2. Include critical context for future AI sessions
3. Document ground truth (production state, not assumptions)
4. Provide quick reference cards and DO/DON'T lists
5. Link to relevant protocol docs (Start-Here-For-AI.md, protocol-v7.md)
6. Archive previous handoffs when creating new ones
7. Keep only the latest handoff active (move older ones to archive/)
