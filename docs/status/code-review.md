## Rules We Enforce
1. **Proof-of-Read**: Every AI response must start with file + quote + rule applied
2. **Green Gate**: `npm run test` + `npm run build` must pass before commit
3. **Scope discipline**: Stay within SCOPE GUARDRAILS defined in prompt
4. **Hot file analysis**: Analysis-only prompt required before changing hot files
5. **Deterministic tests**: No unseeded randomness, no "rerun until green"
6. **Small steps**: ≤ ~50 LOC and ≤ 2 files per step unless explicitly allowed
